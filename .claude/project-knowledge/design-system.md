# Design System

- **Philosophy**: "Cinematic Luxury" (Minimalist, Editorial)
- **Typography**:
  - Primary (UI): Inter
  - Headings (Display): Space Grotesk / Outfit
- **Color Palette**:
  - Background: Charcoal Gray (`#03060C`)
  - Accent: Metallic Gold (`#B8860B`)
- **Tokens**:
  - Rounded Corners: `lg` to `2xl`
  - Spacing: Generous (Embrace whitespace)
  - Interactive States: High-contrast hover states
