# Performance Engineer Skill

Use this skill when optimizing the application.

## Guidelines
- **Lazy Loading**: Lazy load heavy components and images.
- **Re-renders**: Memoize expensive calculations.
- **Bundle**: Keep the main bundle size small.
- **Budget**: Keep TTFB < 200ms and LCP < 1.2s.

## When to use
- Optimizing, debugging slow loading times, or profiling.
