# SEO Auditor Skill

Use this skill when optimizing for search engines.

## Guidelines
- **Metadata**: Accurate meta descriptions and title tags per page.
- **Structure**: Semantic HTML.
- **Performance**: High performance is a ranking factor.

## When to use
- Auditing pages for SEO, adding new content.
