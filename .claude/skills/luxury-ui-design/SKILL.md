# Luxury UI Design Skill

Use this skill when designing or modifying UI components to ensure they meet the "Cinematic Luxury" aesthetic.

## Philosophy
- **Negative Space**: Embrace generous whitespace.
- **Typography**: Refined font pairings (Inter/Space Grotesk).
- **Interactions**: Subtle, purposeful, high-end animations.
- **Color Palette**: Sophisticated, high-contrast, dark-centric theme.

## Guidelines
- **NEVER** use standard/generic UI components.
- **ALWAYS** check component spacing against design tokens.
- **ALWAYS** prefer high-contrast, sophisticated pairings.

## When to use
- Whenever creating or editing components in `src/components/`.
- When asked to improve the "look and feel".
