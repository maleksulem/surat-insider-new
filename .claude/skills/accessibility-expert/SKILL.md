# Accessibility Expert Skill

Use this skill to ensure all users can interact with the site.

## Guidelines
- **Touch Targets**: Minimum 44px x 44px for interactive elements.
- **Aria Labels**: Mandatory for custom icons/buttons.
- **Contrast**: Sufficient color contrast (WCAG 2.1 AA/AAA).

## When to use
- UI/UX review, adding new features.
