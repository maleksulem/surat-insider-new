# Luxury UI Skill

Use this skill when designing or modifying UI components.

## Philosophy
- **Negative Space**: Embrace generous whitespace.
- **Typography**: Refined font pairings.
- **Interactions**: Subtle, purposeful animations.
- **Color Palette**: Sophisticated, high-contrast.

## When to use
- Whenever creating or editing components in `src/components/`.
- When asked to improve the "look and feel".
